# ドットーレ bot コマンド説明

---

## 基本

### `!help`
コマンド一覧をボットが返答します。

---

## プロフィール

ドットーレはあなたの情報を記録・観察しています。
登録しておくと会話の中で反映されます。

### `!profile`
自分のプロフィールシートを表示します。

```
!profile
```

### `!profile set 呼び名 [名前]`
ドットーレがあなたを呼ぶときの名前を登録します。

```
!profile set 呼び名 アリス
```

### `!profile set 備考 [内容]`
自由に備考を記入できます。近況・状態・伝えたいことなど。

```
!profile set 備考 最近睡眠不足気味
```

---

## 観察記録

### `!base`
プロフィールチャンネルに投稿した基本情報の原文を表示します。

```
!base
```

### `!memory`
ドットーレがあなたとの会話から抽出・記録した記憶メモを表示します。

```
!memory
```

### `!nani`
雑談チャンネルで観察・記録された内容をドットーレが報告します。

```
!nani
```

---

## 会話

### `!reset`
ドットーレとの会話履歴をリセットします。
記憶メモやプロフィールは消えません。会話の流れだけリセットされます。

```
!reset
```

---

## VC（音声チャンネル）

### `!kanshi`
ドットーレをあなたが入っているVCに召喚します。
VCに入っていない場合はチャンネル名を指定できます。

```
!kanshi
!kanshi 観測室
```

### `!hakase [メッセージ]`
VC接続中に、指定したテキストに対応する音声をVCで再生させます。

```
!hakase おはよう
```

### `!owari`
ドットーレをVCから退出させます。

```
!owari
```

### `!status`
ドットーレの現在の状態（機嫌・VC接続状況・応答率など）を確認できます。

```
!status
```

---

## 補足

- コマンド以外のメッセージはすべてドットーレへの発言として扱われます。
- プロフィールチャンネルに自己紹介を投稿すると、自動でデータが登録されます。
- ドットーレは全チャンネルの発言を観察しています。
